## Define, then use

`keyframes()` registers the `@keyframes` rule and returns a token whose string value is the animation name. Drop it straight into `animation`.

```ts
const fadeIn = keyframes('fade-in')
  .from({ opacity: 0 })
  .to({ opacity: 1 });

select('.card').style({
  animation: `${fadeIn} 0.4s ease both`,
});
```

Use `from()` / `to()` for two-step animations, or `at()` for explicit percentages:

```ts
keyframes('pulse')
  .at('0%', { transform: 'scale(1)' })
  .at('50%', { transform: 'scale(1.05)' })
  .at('100%', { transform: 'scale(1)' });
```
