## Like media queries, but for a parent

Container queries react to the size of an **ancestor**, not the viewport. First mark the container:

```ts
select('.panel').style({
  containerType: 'inline-size',
  containerName: 'panel',
});
```

Then query it:

```ts
container().name('panel').minWidth('400px').select('.card').style({
  gridTemplateColumns: '80px 1fr',
});
```

Same builder shape as `media()` — `name()`, `minWidth()`, `select()`, and the type hint shows the full `@container` rule.

The panel here uses `resize: horizontal`, so you can **drag its corner** in the preview and watch the card reflow once it crosses 400px — no window resizing needed.
