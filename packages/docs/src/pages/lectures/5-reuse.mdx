## Styles as values

A `style()` is just a bag of properties you can apply anywhere with `use()`. No class names generated, no duplication.

```ts
const button = style({ padding: '0.6rem 1.2rem', borderRadius: '8px' });

select('.save').use(button);
select('.cancel').use(button);
```

`use()` takes as many as you want, applied in order:

```ts
select('.save').use(button, primary);
```

## Building on top

`extend()` merges one style into another:

```ts
const primary = style({ background: '#3498db' }).extend(button);
```

Because it's TypeScript, you can also keep these in their own file and import them wherever you need.
