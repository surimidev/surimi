## Two ways to combine

`and()` makes a **list** (style several selectors at once). `join()` makes a **compound** (one element matching all parts at the same time).

```ts
select('.btn').and('.link');   // .btn, .link
select('.btn').join('.primary'); // .btn.primary
```

## Matching state

`is()`, `not()`, and `has()` take a selector and narrow what you target:

```ts
select('.btn').not('.primary').style({ background: '#eee' });
select('.card').has('img').style({ padding: 0 });
```

## Pseudo-elements

`before()` and `after()` give you `::before` / `::after`. Don't forget `content`:

```ts
select('.tag').before().style({ content: "'#'" });
```
