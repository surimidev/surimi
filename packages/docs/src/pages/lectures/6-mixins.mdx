## style() vs mixin()

A `style()` is just properties. A `mixin()` also carries a **selector fragment** — like `:hover` or `::before` — so it can apply state-based styles when you `use()` it.

```ts
const underlineOnHover = mixin(':hover').style({
  textDecoration: 'underline',
});

select('a').use(underlineOnHover);
// -> a:hover { text-decoration: underline; }
```

The fragment is glued onto whatever you apply it to, so the same mixin works on a link, a button, anything:

```ts
select('button').use(underlineOnHover);
// -> button:hover { ... }
```
